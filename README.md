# Email AI Predictor

Проект для симуляции и анализа эффективности email-рассылок с использованием AI.

## Структура проекта

- `backend/` - Серверная часть на FastAPI (Python)
- `frontend/` - Клиентская часть на React + Vite

## Требования

- Python 3.8+
- Node.js 18+
- LM Studio (или доступ к OpenAI API) для работы AI

## Установка и запуск

### 1. Настройка Бэкенда (Backend)

1. Перейдите в папку бэкенда:
   ```bash
   cd backend
   ```

2. Создайте файл конфигурации из примера:
   ```bash
   cp .env.example .env
   ```
   *Отредактируйте `.env`, если нужно изменить настройки (например, адрес LLM).*

3. Создайте виртуальное окружение и установите зависимости:
   ```bash
   python3 -m venv venv
   ./venv/bin/pip install -r requirements.txt
   ```

4. Запустите сервер:
   ```bash
   ./run.sh
   ```
   Сервер будет доступен по адресу: `http://localhost:8000`

### 2. Настройка Фронтенда (Frontend)

1. Откройте новый терминал и перейдите в папку фронтенда:
   ```bash
   cd frontend
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите режим разработки:
   ```bash
   npm run dev
   ```
   Приложение откроется по адресу: `http://localhost:5173`

## Использование

1. Откройте веб-интерфейс.
2. Введите тему и текст письма.
3. Выберите целевую аудиторию.
4. Нажмите "Запустить симуляцию".
5. AI сгенерирует реакции разных персон и предоставит аналитику.
